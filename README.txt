Instructions for Installing the "wt.rpa" File in Hard to Love

Follow these steps carefully to ensure the patch is installed correctly:

1. Locate the downloaded file "wt.rpa".
2. Extract (unzip) the file if it is in a compressed format (e.g., .zip, .rar).
   - If you do not have a tool to unzip files, you can use software like WinRAR, 7-Zip, or any other extraction tool available for your operating system.
3. After extracting, you should have the "wt.rpa" file ready to use.

Important:

- The "wt.rpa" file must be placed inside the /game/ folder of the Hard to Love game.
- DO NOT place the "wt.rpa" file in the same directory as the game's executable (EXE) file.

---

Visual Guide for PC:

- Correct Path: C:\Program Files (x86)\Hard to Love\game\wt.rpa
- Incorrect Path: C:\Program Files (x86)\Hard to Love\wt.rpa

---

Visual Guide for Mac:

1. Locate the Hard to Love game application in your Applications folder or where it is installed.
2. Right-click on the game application and select "Show Package Contents".
3. Navigate to the following path:  
   Contents/Resources/autorun/game/
4. Place the wt.rpa file inside this game folder.

- Correct Path: /Applications/Hard to Love.app/Contents/Resources/autorun/game/wt.rpa
- Incorrect Path: /Applications/Hard to Love.app/Contents/Resources/autorun/wt.rpa
