# Hard to Love v0.28 - Bugfix Patch

This patch fixes several issues found in Hard to Love v0.28.

## What Gets Fixed

- **Episode 6**: Text display error during Madison's dialogue has been corrected
- **Episode 4**: Image display issue with Nana/Madison scenes now shows correctly
- **Music Player**: Player controls now work properly without errors

## Installation

### ⚠️ BACK UP YOUR ORIGINAL FILE FIRST

Before installing, make a backup copy of your `scripts.rpa` file!

### For Windows

1. Download the new `scripts.rpa` file (~180 MB)
2. Go to your game folder (where you installed Hard to Love)
3. Open the `game` subfolder
4. **Backup**: Copy `scripts.rpa` to another location (or rename it to `scripts.rpa.backup`)
5. Replace the old `scripts.rpa` with the new one from this patch
6. Launch the game

**Example path**: `C:\Program Files\HardtoLove\game\scripts.rpa`

### For Mac

1. Download the new `scripts.rpa` file (~180 MB)
2. Find your Hard to Love app
3. Right-click → "Show Package Contents"
4. Navigate to: `Contents/Resources/autorun/game/`
5. **Backup**: Copy `scripts.rpa` to another location (or rename it to `scripts.rpa.backup`)
6. Replace the old `scripts.rpa` with the new one from this patch
7. Launch the game

**Example path**: `HardtoLove.app/Contents/Resources/autorun/game/scripts.rpa`

## Important Notes

- The file **must** be named `scripts.rpa` (nothing else!)
- Do **NOT** extract or unpack the file
- Place it in the `game` folder (not the main game folder)
- This patch is for **v0.28 only**
- Your save files will continue to work

## If Something Goes Wrong

1. Close the game completely
2. Delete the new `scripts.rpa` file
3. Restore your backup by renaming it back to `scripts.rpa`
4. The game will work as it did before

- **Name**: `scripts.rpa`
- **Size**: ~180 MB
- **Location**: Inside the `game` folder
- **For version**: v0.28
